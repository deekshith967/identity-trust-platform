# Identity Trust Platform

AI-powered Identity Trust and Risk Assessment Platform for Secure Digital Banking.

## Overview

This project is being developed for the Bank of Baroda Hackathon 2026.

The proposed solution continuously evaluates customer identity by analyzing multiple trust signals such as user behaviour, device trust, login history, transaction patterns, location changes, and account recovery activities. An AI-based risk engine generates a dynamic trust score and triggers additional verification only when the risk level is high.
